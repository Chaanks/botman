## Architecture
```
┌───────────────────────────────────────────────────────┐
│                    TEXTUAL TUI                        │
│  Dashboard | Task Queue | Logs | Command Input        │
└────────────────────────────┬──────────────────────────┘
                             │
                    Subscribes/Publishes
                             │
┌────────────────────────────▼──────────────────────────┐
│                   PUBSUB BROKER                       │
│  Topics: orchestrator.command | bot.*.message | ui.*  │
└──┬─────────────────────────────────────┬──────────────┘
   │                                     │
   │                                     │
┌──▼────────────┐                    ┌───▼─────────┐
│ Orchestrator  │                    │  BankActor  │
│    Actor      │                    │             │
└───────────────┘                    └─────────────┘
       │
       │ Assigns tasks
       │
┌──────▼───────────────────────────────────────────────────┐
│                     BOT ACTORS                           │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │  Bot_0   │  │  Bot_1   │  │  Bot_2   │  │  Bot_N   │  │
│  └─────┬────┘  └─────┬────┘  └─────┬────┘  └─────┬────┘  │
└────────┼─────────────┼─────────────┼─────────────┼───────┘
         │             │             │             │
         └─────────────┴─────────────┴─────────────┘
                                 │
                                 │
                                 │
                    ┌────────────▼─────────────┐
                    │    GAME API              │
                    └──────────────────────────┘
┌──────────────────────────────────────────────────────────┐
│                SHARED READ-ONLY DATA                     │
│    GameDataCache | World (Maps, Monsters, Items)         │
└──────────────────────────────────────────────────────────┘
```
